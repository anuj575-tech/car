# Dodge Challenger Showcase

This repository contains the Dodge Challenger showcase website with a custom image-sequence scroll animation.

## Deploying to Vercel

To deploy this project on Vercel:
1. Import this repository in Vercel.
2. Under **Project Settings**, open the **Build & Development Settings**.
3. Set the **Root Directory** to `stitch_showcase`.
4. Click **Deploy**.

Vercel will automatically serve the `index.html` and static assets from the `stitch_showcase` folder.
